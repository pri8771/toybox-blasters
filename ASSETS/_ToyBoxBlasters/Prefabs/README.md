# Prefabs — ToyBox Blasters

Gameplay prefabs for levels, player, enemies, and UI belong here.

**Placeholder art prefabs** are generated under:

- `Assets/_ToyBoxBlasters/Art/Prefabs/Placeholders/`
- `Assets/_ToyBoxBlasters/UI/Prefabs/Placeholders/`

Run **ToyBox Blasters → Setup Placeholder Art** to create or refresh them.
